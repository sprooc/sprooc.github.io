互联网计算-名词解释

英文简称 英文全称 中文 解释
ABR area border routers 区域边界路由器 在使用OSPF的网络中，拥有连接不同区域的接口的路由器，通常是Area 0 和其他区域。
Access links 接入链路 交换机上的链路，只属于一个VLAN
ACK acknowledgement 确认字符 在有确认的传输中，接收方发送给发送方的一种控制字符，表示发来的数据与收到
ACL access control list 访问控制列表 路由器或交换机接口的指令列表，控制端口进出数据包。
ad-hoc Network 自组网络 属于一次对话的各帧
Administrative Distance 管理距离 ⼀个0-255的值，提供路由可靠性的⼀个可选参数
ADSL asymmetric digital subscriber line 非对称数字用户线 上行和下行宽带不对称，利用频分服用，用电话线提供宽带服务
AMI alternate mark inversion 双极性传号交替反转码 n零电平表示“0”，正负电平的跃迁表示“1”，实现对“1”电平的交替反转。
AP access point 接入点 允许无线设备（如笔记本电脑、智能手机、平板电脑等）通过Wi-Fi连接到有线局域网络
ARP address resolution protocol 地址解析协议 根据IP地址查询MAC地址的协议。
ARQ automatic repeat request 自动重传请求 重传的请求是自动进行的。接收方不需要请求发送方重传某个出错的分组。
ASP AppleTalk session protocol AppleTalk会话协议
Asymmetric Switching 非对称交换 交换机不同端口宽带不同
ATM Asynchronous transfer mode 异步传输模式 ATM是以信元为基础的一种分组交换和复用技术。
ATU Access Termination Unit 接入端接单元 在用户端与数字通信网络之间充当接入终端的角色。
ATU-C Access Termination Unit-Central Office 端局接入端接单元 电话端局所用的ADSL调制解
ATU-R Access Termination Unit-Remote 远端接入端接单元 用户端所用的ADSL调制解
axial
backbone 主干 用于vlan间通信
bandwidth 带宽
baud 波特率 信号每秒变化的次数
BDR backup designated router 备份指定路由器 对DR的备份，DR失效时，BDR成为DR
BID bridge ID 网桥ID 两字节的优先级加上6字节的MAC地址，用于选举root bridge 和root port
bit 比特率 每秒传送的二进制位数
BOOTP bootrap protocol 引导程序协议 向BOOTP服务器请求分配IP
BPDU bridge protocol data unit 桥接数据单元 STP帧，用于确定生成树拓扑
BRI basic rate interface 基本速率接口 2B + 1D
bridge 网桥 对收到的帧根据其MAC 帧的目的地址进行转发和过滤
broadcast 广播 一个数据报从源站发送到网络上所有节点
BS basic station 基站 用于提供与移动设备（如手机）之间的通信。
BSS basic service set 基本服务集 一组互相有联系的无线设备
CDM code division multiplexing 码分复用 靠不同编码来区分各路原始信号源的复用方式
CDMA code division multiplex access 码分多址
CHAP challenge handshake authentication protocol 挑战握手认证协议 链路建⽴阶段结 束之后，认证者向对端点发送“challenge”消息;对端点⽤经过单向哈希函数 计算出来的值做应答; 认证者根据它⾃⼰计算的哈希值来检查应答，如果值匹配，认证得到承认， 否则连接应该终⽌; 经过⼀定的随机间隔，认证者发送⼀个新 challenge 给端点，重复步骤 1- 3
CIDR classless inter domain routing 无类域间路由 使用IP地址和子网掩码来确定网络范围，消除了ABC类地址的划分
collision domain 冲突域 局域网（LAN）上，当两个或多个设备同时发送数据帧（数据包）时，可能发生冲突的区域。
CPE Customer Premises Equipment 用户驻地设备 位于用户驻地的设备和内部布线，用户驻地设备连接到运营商的电信信道。用户可以从服务提供商处购买 CPE 或租用 CPE。这里的用户是指从服务提供商或运营商订购 WAN 服务的公司。
CRC cyclic redundancy check 循环冗余校验 选取一个合适的除数，要进行校验的数据是被除数，然后做模2除法，得到的余数就是CRC校验值。
crossover cable 交叉线 一端T568A一端T568B，连接同层设备（思科认为PC网卡是第三层，所以PC连路由器用交叉线）
CSMA/CA carrier sense multiple access with collision avoid 带碰撞避免的载波监听多路复用 发送站点在发送数据前，以控制短帧刺激接收站点发送应答短帧，使接收站点周围 的站点监听到该帧，从⽽在⼀定时间内避免数据发送。⽤于⽆线局域⽹⽐较多
CSMA/CD carrier sense multiple access with collision detection 带碰撞检测的载波监听多路复用 先监听线路，如果发现空闲，则发送数据，否则等待。 在传输过程中，仍然监听线 路，如果冲突检测到，⽴即停⽌传输，则先⼴播拥塞信号，退避算法决定哪个设备再次接⼊介 质。⽤于以太⽹⽐较多
CSU/DSU channel service unit/data service unit 信道服务单元/数据服务单元
Cut-through 直通交换 不校验就转发
DBD Database Description 数据库描述 OSPF中，描述本地LSDB中每一条LSA的摘要信息。
DCE Data Circuit-Terminating Equipment 数据电路终端设备 DCE 由将数据放入本地环路的设备组成。DCE 主要提供一个接口，用于将用户连接到 WAN 网云上的通信链路。
default gateway 默认网关 指在一个子网内，用于转发数据包到其他网络或子网的设备。
DHCP dynamic host configuration protocol 动态主机配置协议 是集中的管理、分配IP地址，使client动态的获得IP地址、Gateway地址、DNS服务器地址等信息，并能够提升地址的使用率。
DIFS distributed inter-frame space 分布协调功能帧间间隔 CSMA/CA中，如果在DIFS时间间隔内信道保持空闲，设备可以发送数据。
DMT Discrete Multi-Tone 离散多音调调制技术 实现ADSL的技术，DMT 调制技术采用频分复用的方法，把 40 kHz 以上一直到 1.1 MHz 的高端频谱划分为许多的子信道，其中 25 个子信道用于上行信道，而 249 个子信道用于下行信道。
DNS domain name system 域名系统 域名系统（DNS）是网络上的一项服务，负责管理域名并响应来自客户端的请求，将域名翻译成相应的IP地址。
Domain Name 域名 由一串用点分隔的名字组成的互联网上某一台计算机或计算机组的名称
DR designated router 指定路由器 在使用OSPF的多路复用网络中，被选举作为代表网络内所有路由器的路由器。
DS distribution system 分配系统 一组相互连接的无线局域网，它们通过一个或多个接入点（Access Point，AP）连接到有线局域网（Local Area Network，LAN）或其他网络。
DSLAM Digital Subscriber Line Access Multiplexer 数字用户线接入复用器 DSLAM是各种DSL系统的局端设备，属于最后一公里接入设备（the last mile），其功能是接纳所有的DSL线路，汇聚流量，相当于一个二层交换机。其在ADSL系统中的位置。
DSSS Direct Sequence Spread Spectrum 直接序列扩频 直接序列扩频通过利用高速率的扩频序列在发射端扩展信号的频谱，而在接收端用相同的扩频码序列进行解扩，把展开的扩频信号还原成原来的信号。
DTE Data Terminal Equipment 数据终端设备 传送来自客户网络或主机计算机的数据以便在 WAN 上传输的客户设备。DTE 通过 DCE 连接到本地环路
DVP distance vector protocol 距离向量协议 通过计算⽬标路由器与源路由器之间的距离 ⽮量和来选择最佳路径，有频繁和周期性的更新，每次更新都将整张路由表发给周围的路由器，
dynamic routes 动态路由 使用路由协议动态更新路由信息。
Dynamic VLAN 动态VLAN 当新的站点连接到一个端口时，交换机通过查表来动态设置端口所属的VLAN
EGP exterior gateway protocol 外部网关协议 用于在不同自治系统（AS，Autonomous System）之间进行路由选择的协议。
EIGRP enhance interior gateway route protocol 增强的内部网关路由协议 Cisco的LSP和DVP的混合协议
ESS extended service set 扩展服务集 由多个使用相同SSID的BSS组成，目的是扩展无线网络覆盖范围。
Fast Forward Switching 快速转发交换 只检查帧的目的MAC，然后立即转发此帧。
FCS frame check sequence 帧校验序列 用于对帧进行校验的字段。
FDDI fiber distributed data inteface 光纤分布式数据接口 一种局域网数据传输标准，以光纤为主要介质。
FDM frequency division multiplexing 频分服用 用户在分配到一定的频带后，在通信过程中自始至终都占用这个频带。
Flow control 流控制 一种保证不在缓冲区已满的情况下发送端继续传输数据的机制。
Fragment Free 免碎片 只读取前64个字节来检测错误，然后转发
Frame Filtering 帧过滤
Frame Relay 帧中继 帧中继（Frame Relay）是一种在计算机网络中用于在广域网（WAN）上进行数据传输的数据链路层协议。它提供了一种简单而高效的方法，允许在不同地理位置的网络设备之间进行数据通信。
Frame Tagging 帧标记
FTP file transfer protocol 文件传输协议 用于在网络上进行文件传输的一套标准协议，它工作在 OSI 模型的第七层， TCP 模型的第四层， 即应用层， 使用 TCP 传输而不是 UDP， 客户在和服务器建立连接前要经过一个“三次握手”的过程， 保证客户与服务器之间的连接是可靠的， 而且是面向连接， 为数据传输提供可靠保证。
FTTx Fiber to the x 光纤到x HTTH,光纤到家 HTTB光纤到大楼, HTTC光纤到路边
full-duplex transmission 全双工 信号可以同时双向传输
GIF graphic interchange format 图形交换格式 一种公用的图像文件格式标准
half-duplex transmission 半双工 信号双向传输，但不能同时传输
HDLC high-level data link control 高级数据链路控制 在同步⽹上传输数据、⾯向⽐特的数 据链路层协议。思科路由器上的默认WAN接⼝封装协议。
HDSL High Speed DSL 高速数字用户线
HFC Hybird Fiber Coax 光线同轴混合网 HFC 网是在目前覆盖面很广的有线电视网 CATV 的基础上开发的一种居民宽带接入网。
hold-dowm timer 挂起计时器 路由器需要将某些可能导致路由环路的网络状态的变化保留一段时间，在这段时间内，路由器将视情况对这些网络状态的变化所产生的路由信息进行更改。
HTML hyper-text mark language 超文本标记语言 一种用于创建网页的标准标记语言。
HTTP hyper-text transfer protocol 超文本传输协议 客户端和服务端进行数据传输的一种规则
HTTPD HTTP daemon 超文本传输协议守护程序 在万维网服务器等待 HTTP 请求的进程
hub 集线器 对接收到的信号进行再生整形放大，以扩大网络的传输距离，同时把所有节点集中在以它为中心的节点上。
Hybird Protocol 混合协议 DVP和LSP的混合协议。
ICMP Internet control message protocol 网际控制报文协议 是为了提⾼IP数据报交付成功的机会，允许主机或路由器报告差错情况和提供有关异常情况报告的协议，运⾏在IP层
IDSL ISDN DSL ISDN数字用户线
IEEE institute of electrical and electronic engineers 电气和电子工程师协会
IGP interior gateway protocol 内部网关协议 用于在单一自治系统（AS，Autonomous System）内部进行路由选择的协议。
IGRP Interior gateway routing protocol 内部网关路由协议 思科的一种DVP
infrared 红外线
ISDN integrated service digital network 混合业务数字网络 一种电路交换技术，能够让 PSTN 本地环路传输数字信号，从而实现更高容量的交换连接。
IS-IS intermediate-system to intermediate-system 中间系统到中间系统 LSP和DVP的混合协议
ISL Inter-Switch Link 交换机间链路 思科专利，封装帧，加上VLAN信息
ISO
ISP internet service provider 互联网服务提供商 向广大用户综合提供互联网接入业务、信息业务、和增值业务的电信运营商。
JPEG joint photographic experts group 联合图像专家组 一种针对照片影像而广泛使用的有损压缩标准方法
LAPB Link Access Procedure Balanced 链路接入规程，平衡式 链路访问过程平衡（LAPB）是数据链路层协议，负责管理在 X.25 中 DTE 设备与 DCE 设备之间的通信和数据包帧的组织过程。
LAPD Link Access Procedure On the D-Channel D信道链路接入规程 一种面向连接、可靠的数据链路层协议，通常用于数字通信网络，特别是在 ISDN（Integrated Services Digital Network）和 X.25 等环境中。它负责在 D 通道上建立、维护和释放连接，并处理信令、控制信息，以及错误检测，以确保在用户设备和网络交换设备之间的数据传输的可靠性和有序性。
laser 激光
late collision 迟冲突 64byte帧发送后发生的错冲突，它将影响网络性能。在一个健康的网络中极少发生此类冲突。
LCP link control protocol 链路控制协议 在PPP协议中，用于建立连接，配置连接选项，测试链路质量
line coding 线路编码 把二进制信息转化为可以在物理介质上传输的形式。
link 链路 连接两个网络设备的物理连接
LLC logical link control 逻辑链路控制 与网络层交互，标识不同的协议类型并封装它们
LSA link-state advertisements 链路状态通告 装载和传输链路状态信息的报文
LSAck Link-State Acknowledgement 链路状态确认 OSPF中，对收到的LSA进行确认
LSP link state protocol 链路状态协议 每个路由器都了解整个⽹络的拓扑结构，利⽤算法 计算两个路由之间的最短路径，更新由事件触发，每次更新都只向周围的路由器传递路由表的更 新信息，包括OSPF等
LSR Link-State Request 链路状态请求 OSPF中，两个路由器交换DBD后，发送LSR请求所有LSA
LSU Link-State Update 链路状态更新 OSPF中，向对方发送所需要的LSA
MAC madia access control 介质访问控制 与物理层交互，定义如何在物理层上传输帧
MIME Multipurpose Internet Mail Extension 通用互联网邮件扩充 MIME 在其邮件首部中说明了邮件的数据类型（如文本、声音、图像、视像等）。在MIME 邮件中可同时传送多种类型的数据。
Modems 调制解调器 负责模拟信号和数字信号之间的转化
MSL max segment lifetime 最长报文段生存时间
MSS maximun segment size 最长报文段长度 MSS 告诉对方缓存所能接收的报文段的数据字段的最大长度是 MSS 个字节
MTU maximum transmission unit 最大传送单元 指网络能够传输的最大数据包大小，以字节为单位。一般由数据链路层规定。（以太网1500）
multicast 组播 一个数据报从源站发送到特定的一组目的站
NAT network address translation 网络地址转换 是⼀种把内部私有⽹络地址（IP地址）翻译成外部⽹络IP地址的技术
native VLAN 本征VLAN 用于未标记帧的默认VLAN
NAV network allocation vector 网络分配向量 CSMA/CA中，指出了信道处于忙状态的持续时间。
NCP network control protocol 网络控制协议 PPP协议中，选择和配置第三层协议
NFS netword file system 网络文件系统 一种分布式文件系统，力求客户端主机可以访问服务器端文件，并且其过程与访问本地存储时一样
NIC netword interface card 网卡 网卡是一块被设计用来允许计算机在计算机网络上进行通讯的计算机硬件。由于其拥有MAC地址，因此属于OSI模型的第1层和2层之间。它使得用户可以通过电缆或无线相互连接。
NRZ non return to zero 不归零制码
NVRAM non-volatile RAM 非易失随机存取存储器
OC Optical Carrier 光载波
ODN Optical Distribution Node 光纤分配节点 HFC网中，在光纤结点光信号被转换为电信号。在光纤结点以下就是同轴电缆。
OSI reference model open system interconnection reference model 开放系统互联参考模型 国际标准化组织（ISO）提出的网络模型，有助于网络构建者实现可以相互通信和协同工作的网络。描述了信息或数据如何从一个计算机通过网络传输到另一台计算机的过程，采用分层通信的方法，每个层次执行特定任务。
OSPF open shortest path first 开放最短路径优先 LSP的一种实现，每个路由器维护一个描述整个自治系统链路状态的数据库， 通过最短路径算法求出最短路径树，从而决定最优路由。
OUI organizational unique identifier 组织唯一标识符 生产局域网适配器的厂家都必须向IEEE 购买由这三个字节构成的这个号，用于标识其生产商
PAP password authentication protocol 口令鉴别协议 远程节点不停的在链路上反复发送⽤ 户名/密码，直到验证通过或者连接终⽌。不健壮的身份认证协议，使⽤明⽂发送密码。连接建 ⽴前只有⼀次认证
PAT port address translation 端口地址转换 是对⽹络地址转换（NAT）的扩展，它允许局域⽹（LAN）上的多个设备映射到⼀个单⼀的公共IP地址的不同端口。
PDU protocol data unit 协议数据单元 对等层次之间传递的数据单位
PID Port ID 端口ID
ping packet internet groper 使用ICMP的会送请求和应答报文来测试两个主机之间的联通性的应用。
POP post office protool 邮局协议 用于支持使用客户端远程管理在服务器上的电子邮件
POST Power On Self Test 开机自检软件 储存在路由器的ROM，开机时运行，用于检查硬件。
PPP point to point protocol 点对点协议 通过拨号或专线⽅式在两个⽹络节点之间建⽴连 接、发送数据。提供对多种⽹络层协议的⽀持。
preamble 前同步码 以太网帧前用于实现位同步的8个字节同步码，交替的1和0
PRI primary rate interface 主速率接口 T1:: 23B + D; E1: 30B + D
Proxy ARP 代理ARP
PS POTS Splitter 电话分离器
PSTN Public Switch Telephone Network 公共交换电话网 日常生活中的电话网
PVC Permanent Virtual Circuits 永久虚电路 信双方的电路在用户看来是永久连接的虚电路。
radio 无线电
RADSL Rate Adaptive DSL 速率自适应DSL ADSL 的一个子集，可自动调节线路速率
RAM random access memory 随机存取存储器
RARP reverse address resolution protocol 逆向地址解析协议 ARP为IP到MAC的转换，而RARP为MAC到IP的转换，向RARP服务器请求分配IP。
repeater 中继器 适用于完全相同的两个网络的互连，主要功能是通过对数据信号的重新发送或者转发，来扩大网络传输的距离。 中继器是对信号进行再生和还原的网络设备：OSI模型的物理层设备。
RIP router information protocol 路由信息协议 一种分布式的基于距离向量的路由选择协议，是互联网的标准协议，其最大优点就是简单。
rollover cable 反转线 控制台电缆，全部插脚相反，用于将工作台连接到路由器或交换机进行配置。
route aggregation 路由聚合 路由聚合（也叫汇总）是让路由选择协议能够用一个地址通告众多网络，旨在缩小路由器中路由选择表的规模，以节省内存，并缩短IP对路由选择表进行分析以找出前往远程网络的路径所需的时间。
Route poisoning 路由毒害 防止路由环路的一种技术，当路由器发现一个网络不可达，不是将路由条目删除，而是将距离设为无穷大，让其它路由器直到这一网络不可达。
router 路由器 工作在第三层的网络设备，对不同网络之间的数据报进行路径选择、分组交换
routing protocol 主动路由协议 主动路由协议确定被动路由协议到达目的地的路径。
RPC remote procedual call 远程过程调用 一种通过网络从远程计算机程序上请求服务，而不需要了解底层网络技术的协议。
RTS request to send 请求传输帧 RTS 是一种用于协调无线网络中节点之间数据传输的信号。当一个节点准备发送数据时，它发送 RTS 信号以请求发送数据的权限。
RZ return to zero 归零制码
SCP DNA secion control protocol 会话控制协议
ScTP screened twisted pair 网屏式双绞线
SDH Synchronous Digital Hierarchy 同步数字系列 ITU-T 以美国标准 SONET 为基础，制订出国际标准同步数字系列 SDH
SDSL Single-line DSL 一对线的数字用户线
SIFS short inter-frame space 短(Short)帧间间隔 分隔开属于一次对话的各帧
simplex transmission 单工 信号单向传输
SLIP serial line internet protocol 串线网际协议 广域网第二层协议，一种在串行线路上对IP数据报进行封装的简单形式
SMTP simple mail transfer protocol 简单邮件传输协议 可用在发送和接收电子邮件的信息
SNMP simple network management protocol 简单网络管理协议 一种应用层协议，用于在不同网络设备之间交换管理信息。
socket 套接字 对网络中不同主机上的应用进程之间进行双向通信的端点的抽象。包括IP地址和端口号
SONET Synchronous Optical Network 同步光纤网 美国定义的数字传输标准，各级时钟都来自一个非常精确的主时钟
Source routing Bridge 源路由网桥 在发送帧时将详细的路由信息放在帧的首部中
SPF shortest path first 最短路径优先
split horizen 水平分割 解决路由环路的方法之一，路由器从某个端口接收到的更新信息不再从这个端口发出去。
SQL structrued query language 结构化查询语言 一种特定目的编程语言，用于管理关系数据库管理系统
SRA Seamless Rate Adaption 无缝速率自适应技术 ADSL2使用的技术。在运营中不中断通信和不产生误码的情况下，自适应地调整数据率。
SSID service set identifier 服务集标识符 对BSS的标识
static routes 静态路由 网络管理员手工输入路由信息。
Static VLAN 静态VLAN 直接指派端口所属的VLAN
STDM static time division mulplexing 统计时分复用 但每一个STDM 帧中的时隙数小于连接在集中器上的用户数。各用户有了数据就随时发往集中器的输入缓存，然后集中器按顺序依次扫描输入缓存，把缓存中的输入数据放入STDM 帧中。对没有数据的缓存就跳过去。当一个帧的数据放满了，就发送出去。因此， STDM 帧不是固定分配时隙，而是按需动态地分配时隙。因此统计时分复用可以提高线路的利用率。
STM-1 Synchronous Transfer Mudule 同步传递模块 SDH 的基本速率为 155.52 Mb/s，称为第 1 级同步传递模块 (Synchronous Transfer Module)，即 STM-1，相当于 SONET 体系中的 OC-3 速率。
Store-and-forward 存储转发 交换机存下整个帧，校验CRC，然后再转发该帧
STP shielded twisted pair 屏蔽双绞线
STP spanning tree protocol 生成树协议 通过计算稳定的生成树网络拓扑来防止环路
straight cable 直通线 两端均是T568A或均是T568B，用于连接跨层设备。
STS-1 Synchronous Transport Signal 第一级同步传送信号 SONET 为光纤传输系统定义了同步传输的线路速率等级结构，其传输速率51.84Mbit/s 为基础，大约对应于T3/E3 的传输速率
stub network 末节网络 仅有一单个通路通过路由器连接到其他子网，或者仅有一个通路连接到其他网络。
SVCs Switched Virtual Circuits 交换虚电路 按需建立和终止的WAN路径
switch 交换机
Symmetric Switching 对称交换
TCP transmission control protocol 传输控制协议 是⼀种⾯向连接的、可靠的、基于字节 流的传输层通信协议
TDM time division multiplexing 时分复用 将时间划分为一段段等长的时分复用（TDM ）帧，每个时分复用的用户在每个 TDM 帧中占用固定序号的时隙。
Telnet 远程登录服务
TFTP trivial file transfer protocol 一般文件传输协议
throuput 吞吐量
TLD top level domain 顶级域名 一级域名，最高层
Transparent Bridge 透明网桥
Trunk links 主干链路 支持多个VLAN通信，用于连接交换机或路由器
UDP user datagram protocol 用户数据报协议 OSI参考模型中⼀种⽆连接的传输层协 议，提供⾯向事务的简单不可靠信息传送服务。
UIB User Interface Box 用户接口盒 HFC网中，每个家庭要安装一个用户接口盒用户接口盒 要提供三种连接，即：使用同轴电缆连接到机顶盒(set-top box)，然后再连接到用户的电视机；使用双绞线连接到用户的电话机；使用电缆调制解调器连接到用户的计算机。
unicast 单播 一个数据报从源站发送到单一目的站
URL uniform resource locator 统一资源定位符 对可以从因特网上得到的资源的位置和访问方法的一种简洁的表示。
UTP unshielded twisted pair 无屏蔽双绞线
VDSL Very high speed DSL 甚高速用户数字线
VLAN virtual local area network 虚拟局域网 是一组逻辑上的设备和用户，这些设备和用户并不受物理位置的限制，可以根据功能、部门及应用等因素将它们组织起来，相互之间的通信就好像它们在同一个网段中一样，由此得名虚拟局域网。用于划分逻辑子网。工作在第二层和第三层。可以分割广播域
VLSM variable length subnet mask 可变长子网掩码 允许一个自治系统内有不同子网掩码的网络。
WAN Wide Area Network 广域网 通过WAN服务提供商连接LAN的通信网络。
WDM wavelength division mulplexing 波分复用 波分复用就是光的频分复用。
Wi-Fi Wireless-Fidelity 无线保真度
xDSL x Digital Subscriber Line xDSL 技术就是用数字技术对现有的模拟电话用户线进行改造，使它能够承载宽带业务
